# Roadmap

## Delivered

1. Native ASP.NET Core 8 Blazor Server migration.
2. Clean Architecture domain, application, infrastructure, and UI layers.
3. MySQL Code First model, repositories, and Identity roles.
4. Merchant registration, profile, KYC upload, and approval tracking.
5. Admin dashboard, application inventory, and review decisions.
6. Secure API credential issuance and rotation.
7. Architecture, setup, security, schema, and interview demo documentation.
8. Automate first-run development admin credentials with ASP.NET Core User
   Secrets while preserving explicit override and reset options.
9. Improve merchant onboarding with required-field indicators, standard
   business type and industry choices, editable profiles, and company logos.
10. Complete KYC and admin workflows with functional milestones, authorized
    document downloads, merchant-aware activity messages, filters, and admin
    navigation context.
11. Add PayPortal branding, light/dark themes, a useful account menu, corrected
    typography and spacing, and role-appropriate navigation.
12. Refresh demo and technical documentation for all feedback changes.

## Next

13. Add unit, integration, authorization, and browser tests.
14. Add production email, private object storage, malware scanning, MFA, CI,
    observability, and deployment configuration.
