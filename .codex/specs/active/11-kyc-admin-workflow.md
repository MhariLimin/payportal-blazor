# 11. KYC and Admin Workflow

## Goal

Make KYC progress, review evidence, activities, and admin navigation complete
and understandable.

## Acceptance Criteria

- Required-document progress updates after uploads and review decisions.
- Milestone descriptions explain what happens and in what order.
- Uploading documents is allowed before compliance review; review begins after
  required evidence is present.
- Authorized merchants and admins can view or download KYC documents.
- Admin dashboard activity identifies the merchant and uses readable titles.
- Application filters include business type and industry.
- Admin profile and KYC detail pages include breadcrumbs and a back action.
- Merchant-only navigation is hidden from admins.
