# 21. Feedback 5 Document Requests

## Outcome

- Merchant document requests are no longer a separate sidebar page.
- Missing and rejected evidence are shown inside the KYC Document Requirements
  card.
- Merchants can submit or resubmit documents from the affected requirement row.
- Rejected documents are visually marked and crossed out.
- Admin Document Requests groups open requests by merchant with expandable
  request lists.
