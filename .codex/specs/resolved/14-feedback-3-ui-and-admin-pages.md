# 14 Feedback 3 UI and Admin Pages

## Goal

Make PayPortal visibly denser and remove confusion between admin application
pages.

## Acceptance Criteria

- Shared typography, list rows, cards, buttons, inputs, and navigation are
  materially smaller.
- Recent Activity item text is smaller than the card heading.
- `/applications` is the review-focused Applications workspace.
- The searchable all-merchant list is renamed Merchant Directory and moved to
  `/merchant-directory`.
- Dashboard chart links open filtered Merchant Directory views.

## Status

Resolved.
