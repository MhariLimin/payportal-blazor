# .codex Directory

This directory is the compact working context system for PayPortal.

- `CODEX.md`: session entry point.
- `commands/`: context-loading procedures.
- `context/`: product, architecture, state, conventions, and roadmap notes.
- `specs/index.md`: spec map.
- `specs/pending/`: planned work.
- `specs/active/`: work currently being implemented.
- `specs/resolved/`: shipped and verified work.
